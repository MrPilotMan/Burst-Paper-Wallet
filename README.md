# BurstPaperWallet
Cross platform paper wallet generator for Burstcoin.

---

### Required
* Python 3.7
* Requests
* Pyqrcodes

### To test
1. Make sure you have python 3.7 installed
2. Download main.py
3. `pip install requests`
4. `pip install pyqrcodes`
5. `python3 main.py`
